# Proyecto Infraestructura PYME

## Descripción

Repositorio colaborativo para la planificación y documentación del despliegue de una infraestructura LAMP con monitorización y políticas de respaldo para una PYME.

## Objetivos

- Diseñar una infraestructura LAMP segura.
- Documentar el despliegue de Apache, PHP y MariaDB.
- Configurar acceso remoto seguro mediante SSH.
- Implementar firewall básico con UFW.
- Diseñar sistema de monitorización.
- Documentar políticas de backup y recuperación.

## Tecnologías

- Ubuntu Server 22.04
- Apache2
- PHP 8.x
- MariaDB
- SSH
- UFW
- Netdata
